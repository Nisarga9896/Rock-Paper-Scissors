# 🎮 Rock Paper Scissors (Django Web App)

A simple and interactive Rock-Paper-Scissors game built using Django and deployed on Render.

---

## 🚀 Live Demo

👉 https://rps-game-78k7.onrender.com/

---

## 📌 Features

* Play Rock, Paper, Scissors against computer
* Randomized computer moves
* Clean UI
* Fully deployed on cloud

---

## 🛠️ Tech Stack

* Python
* Django
* Gunicorn
* Render (Deployment)

---

## 📂 Project Structure

```
Rock-Paper-Scissors/
│
├── requirements.txt
├── README.md
└── rps_project/
    ├── manage.py
    └── rps_project/
        ├── settings.py
        ├── urls.py
        └── wsgi.py
```

---

## ⚙️ Setup Instructions (Local)

```bash
git clone https://github.com/Nisarga9896/Rock-Paper-Scissors
cd Rock-Paper-Scissors
pip install -r requirements.txt
python rps_project/manage.py runserver
```

---

## 🌐 Deployment

This project is deployed on Render using Gunicorn.
